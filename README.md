# Payment Gateway Backend API

A secure and scalable payment gateway backend built with Node.js, NestJS, Javascript and TypeScript. This application provides APIs for transaction initialization, payment processing, transaction verification, and payment gateway integration.

## Features

* Transaction initialization and reference generation
* Secure payment processing workflow
* Paystack & Flutterwave payment integration
* Transaction verification endpoint
* API key authentication
* Currency validation and support
* Callback URL handling
* Secure account and customer validation
* MySQL database integration
* RESTful API architecture
* TypeScript-based modular backend structure

## Technology Stack

* Node.js
* NestJS
* TypeScript
* MySQL
* REST API
* Paystack API Integration

## Core Modules

* Payment Initialization
* Transaction Processing
* Transaction Verification

* Customer Validation
* Gateway Integration

## Security Features

* API key validation
* Secure transaction references

## API Capabilities

* Initiate transactions
* Perform payments
* Verify transaction status
* Generate secure references
* Handle payment redirects
* Process payment callbacks
